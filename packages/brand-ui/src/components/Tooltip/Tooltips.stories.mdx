# Tooltip

`Tooltip` is brief piece of information that appear on hover state over an element to clarify its meaning or use for the user.

<Canvas>
  <Story name="Default tooltip">
    <Tooltip label="Label"><Button>Children component</Button></Tooltip>
  </Story>
</Canvas>

## Visibility

With the `visible` props you can set if the tooltip is visible or not initially.
By default this prop is `false`, so it is hidden initially.

<Canvas>
  <Story name="Visible">
    <Tooltip label="Label" visible={true}><Button>Visible</Button></Tooltip>
  </Story>
  <Story name="Hidden">
    <Tooltip label="Label" visible={false}><Button>Hidden</Button></Tooltip>
  </Story>
</Canvas>

## Placement

If you use the `placement` prop you can set the tooltip placement relative to the tooltip trigger component.
The possible `placement`s are: `top`, `right`, `bottom` and `left`. The default `placement` is `top`.

<Canvas>
  <Story name="Placement top">
    <Tooltip label="Label" placement="top"><Button>Top</Button></Tooltip>
  </Story>
  <Story name="Placement right">
    <Tooltip label="Label" placement="right"><Button>Right</Button></Tooltip>
  </Story>
  <Story name="Placement bottom">
    <Tooltip label="Label" placement="bottom"><Button>Bottom</Button></Tooltip>
  </Story>
  <Story name="Placement left">
    <Tooltip label="Label" placement="left"><Button>Left</Button></Tooltip>
  </Story>
</Canvas>

## Sx

The `tooltip` can receive `sx` props to change it's design, as you can see below with some examples.

<Canvas>
  <Story name="Custom sx">
    <Tooltip label="Label" sx={{backgroundColor: 'warning.base', color: 'black'}}><Button>Button</Button></Tooltip>
  </Story>
</Canvas>
